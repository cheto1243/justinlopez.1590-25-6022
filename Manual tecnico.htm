Manual Técnico del Sitio Web Personal

1. Estructura general del sitio

Se utilizó un diseño basado en <frameset>, el cual permite dividir la pantalla en dos secciones: un menú a la izquierda y un área de contenido a la derecha.

<frameset cols="20%,80%"> <frame src="menu.html"> <frame name="contenido"> </frameset>

2. Menú

El archivo menu.html contiene los enlaces que permiten navegar entre las distintas secciones del sitio. Cada enlace carga una página distinta en el marco llamado contenido.

<ul> <li><a href="hoja.html" target="contenido">Hoja de Vida</a></li> <li><a href="tabla.html" target="contenido">Tabla de Notas</a></li> <li><a href="galeria.html" target="contenido">Galería de Imágenes</a></li> <li><a href="video.html" target="contenido">Video</a></li> <li><a href="manual.html" target="contenido">Manual Técnico</a></li> </ul>

3. Hoja de Vida

Esta sección muestra los datos personales del alumno en formato estructurado.

4. Tabla con notas

Se creó una tabla que muestra los nombres de los cursos cursados y sus notas de primer y segundo parcial.

<table border="1"> <tr><th>Curso</th><th>1er Parcial</th><th>2do Parcial</th></tr> <tr><td>Matemática</td><td>85</td><td>90</td></tr> <tr><td>Informática</td><td>88</td><td>92</td></tr> </table>

5. Galería de imágenes

La galería se diseñó usando imágenes almacenadas localmente en una carpeta llamada imagenes2. A continuación, se muestra cómo insertar varias imágenes.

<img src="imagenes2/galaxia.png" alt="galaxia"> <img src="imagenes2/lago.png" alt="lago"> <!-- Se repite hasta tener al menos 10 imágenes -->

6. Video de YouTube

Para insertar un video desde YouTube se usó un <iframe> como se muestra a continuación:

<iframe width="560" height="315" src="https://www.youtube.com/embed/EvKm8yhM7V8" frameborder="0" allow="autoplay; encrypted-media" allowfullscreen></iframe>

7. Herramientas utilizadas

  • Editor de código: Visual Studio Code
  • Navegador: Google Chrome
  • Lenguajes: HTML, CSS
  • Recursos externos: YouTube

8. Conclusión

Este sitio web fue desarrollado como parte del proceso formativo de la carrera de Ingeniería en Sistemas. Su construcción permitió aplicar conocimientos básicos de HTML y CSS, además de comprender el uso de marcos, tablas, inserción de multimedia y navegación entre páginas.
